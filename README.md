# PCA_Natation
The project was to build a PCA (Principal Component Analysis) model to analyse swimming performance data.

### 📘 Current Features
- **Synthetic dataset** of 80 swimmers generated for exploratory analysis  
- **PCA visualisation** showing the projection of swimmers on the first components  
- **Correlation circle** illustrating the contribution of each variable to the principal components  

These elements allow us to identify dominant performance factors and visualise how athletes group together based on their characteristics.

### 🔮 Future Work
- Import and merge **real competition and training data**  
- Extend the analysis with additional variables and longitudinal data  
- Explore **linear models for each principal component**  
- Compare clusters with real performance outcomes  
- Build an automated pipeline for visualisation and interpretation  

This repository will evolve with real-world swimming data becomes available, enabling deeper performance insights and more advanced modelling
