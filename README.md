# PCA_Natation

### Project Overview
This project performs Principal Component Analysis (PCA) on swimming performance data. It includes data preprocessing, PCA computation, and visualization of results.

### Features
- Synthetic data generation for swimmers
- Data standardization and correlation analysis
- PCA computation with visualizations:
  - Scatter plot of individuals on PC1 x PC2
  - Correlation circle for variables
- Clustering and outlier detection

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Usage
Run the Jupyter notebooks in the `notebooks/` directory to explore the analysis.
